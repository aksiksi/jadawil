Welcome to Jadawil!

This is an extremely early version of my upcoming project. I've deployed this because of the early registration period. Please report any bugs to cyph0nik[at]gmail[fullstop]com. Sorry for the bland design!

Instructions

  1. Enter the course codes of the courses you'd like to take in Spring 2013.
  2. A few examples of what a course code can look like: PSY 300, MATH 1120, MECH 390, ELEC 305. Uppercase or lowercase doesn't matter - just make sure that there's a space in between.
  3. Hit the submit button to get a list of upto 15 possible schedules. If the site returns less than 15 schedules, know that these are the only possible combinations.

Notes

If you get no results, one of the following has occured:

  1. There is a direct conflict between two courses and no alternatives are available.
  2. The scheduler did not find any valid schedules within the 10 second limit.
  3. There are too many possible combinations.
